# CraftKit Roadmap

## Phase 0 — Bootstrap

Goal: create a coherent markdown-first repository with stable terminology.

Deliverables:

- README
- AGENTS
- product notes
- roadmap
- first-pass core skill files
- bootstrap prompts

## Phase 1 — Normalize core skills

Goal: make the six core skills consistent in structure and tone.

Deliverables:

- prompt
- blueprint
- reflect
- tune
- research
- autoresearch
- one example per skill
- failure-mode notes per skill

## Phase 2 — Migrate existing assets

Goal: absorb reusable ideas from older prompt-builder and meta-skill assets.

Deliverables:

- migration notes
- renamed or merged skill files
- deprecated patterns called out explicitly
- terminology alignment

## Phase 3 — Package for reuse

Goal: make CraftKit easy to copy, install, or import into other repos.

Deliverables:

- publish-ready metadata
- license
- concise quickstart
- example usage flows

## Phase 4 — Optional eval layer

Goal: add a lightweight way to compare prompt and skill revisions.

Possible deliverables:

- example eval criteria templates
- sample review loop
- simple scorecard format

## Current priority

Bootstrap first. Do not overbuild before the skill files are clean and reusable.
